---

O Curso React.js Ninja para aqueles desenvolvedores que querem crescer como programadores na rea de desenvolvimento web. Nesse curso ser abordado no s sobre a lib React.js em si, mas todas as ferramentas e tcnicas do ecossistema ao seu redor: redux, react-router, Server Side Rendering, programao funcional, testes, etc.

O curso est em andamento, e est sendo separado por mdulos. Para acompanhar a ementa de cada mdulo, e todos os cdigos utilizados, acesso o repositrio do curso, entrando no GitHub, e acessando /da2k/curso-reactjs-ninja.

O que voc aprender

  • Criar aplicaes utilizando o React.js de forma correta
  • Configurar seu ambiente de desenvolvimento e produo
  • Usar conceitos de programao funcional em suas aplicaes
  • Aplicar testes de vrios tipos em suas aplicaes
  • Criar aplicaes com rotas, estados isolado e pr-renderizao no lado do servidor

H algum requisito ou pr-requisito para o curso?

  • necessrio estar confortvel com a linguagem JavaScript (ao menos na verso ES5)
  • preciso ter um conhecimento bsico da API DOM

Para quem este curso:

  • Desenvolvedores com conhecimento bsico / intermedirio em JavaScript que queiram entender sobre como o ReactJS funciona, e todo o seu ecossistema.
